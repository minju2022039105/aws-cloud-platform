# 모두의 창업 1기 — 아이디어 신청 초안

---

## 아이디어명

**ShieldOps** — 보안팀 없는 중소기업을 위한 AI 보안 관제 자동화 플랫폼

---

## 한 줄 소개

서버를 올린 지 몇 시간 만에 해외 봇이 붙었습니다.
로그는 쌓였지만, 실시간으로 탐지하고 대응할 방법은 없었습니다.
그래서 만들었습니다.

---

## 시작된 이유

직접 AWS 인프라를 구축하고 운영하면서 이상한 걸 발견했습니다.

서버를 연 지 얼마 지나지 않아 해외 IP가 관리자 페이지를 스캔하고, Exchange 취약점을 탐색하고, 인코딩된 XSS 요청을 반복적으로 보내기 시작했습니다. 공격인지도 몰랐고, 알아도 막을 수가 없었습니다.

보안팀이 없는 팀에서는 이게 일상입니다. 공격은 24시간 오는데 사람은 자야 합니다.

---

## 문제

많은 중소기업과 스타트업은 보안 전담 인력이 없습니다.

기존 선택지는 둘뿐입니다.

**WAF만 쓰기**: 알려진 패턴은 막지만, 인코딩을 한 번만 바꿔도 그냥 통과됩니다. 막힌 건지 뚫린 건지도 모릅니다.

**MSSP 계약하기**: 월 수백만 원. 스타트업이 감당하기 어렵습니다.

그 사이 어딘가에 있어야 할 선택지가 없었습니다.

---

## 해결 방법

**ShieldOps**는 세 레이어로 동작합니다.

**1차 — 룰 기반 차단**
GeoBlock, SQLi/XSS 룰, IP Reputation으로 알려진 공격을 즉시 차단합니다.

**2차 — AI 이상 탐지**
Isolation Forest 모델이 URI 구조와 Shannon Entropy를 분석합니다. 룰을 우회한 변형 공격도, 정상처럼 보이는 저속 스캔도 패턴이 다릅니다. 그 차이를 잡습니다.

**3차 — 위험도 기반 자동 대응**
AI가 이상을 감지하면 위험도에 따라 단계적으로 대응합니다. 고위험 IP는 즉시 차단하고, 중간 위험은 운영자에게 알림을 보냅니다. 새벽 3시에 공격이 와도 사람이 깨어있을 필요가 없습니다.

---

## ShieldOps만의 차별점 — ISMS 자동 점검

보안 공격 대응 외에, ShieldOps는 ISMS 컴플라이언스 자동 점검 기능을 함께 제공합니다.

ISMS 인증은 중소기업이 가장 어려워하는 숙제입니다. 무엇을 점검해야 하는지, 지금 어떤 항목이 위반 상태인지 파악하는 것만으로도 전문 컨설팅 비용이 발생합니다.

ShieldOps는 AWS Config Rules로 ISMS 통제항목 11개를 상시 자동 점검합니다.
루트 키 노출, MFA 미설정, S3 공개 접근, CloudTrail 무결성 등 위반 즉시 알림을 보냅니다. 별도 컨설팅 없이 지금 당장 현황을 파악할 수 있습니다.

공격 방어와 인증 준비를 하나의 플랫폼에서 해결합니다.

---

## 지금 어디까지 왔나요?

아이디어가 아닙니다. 지금 이 순간도 돌아가고 있습니다.

**✅ 완료**
- WAF 5단계 룰 + Terraform 코드화
- Isolation Forest AI 탐지 엔진 개발 및 재학습
- Lambda SOAR 자동 차단 파이프라인
- ISMS Config Rules 11개 자동 점검 파이프라인
- GitHub Actions CI/CD + tfsec 보안 게이트

**🔄 운영 중**
- 실제 인프라에서 해외 봇 트래픽 실시간 탐지 및 차단
- Athena 기반 WAF 로그 분석 파이프라인

**🔧 고도화 예정**
- Grafana 보안 관제 대시보드 (룰별 차단 현황, AI 이상 점수 분포, 국가별 히트맵)
- 오탐률 추이, MTTD/MTTR 운영 지표

---

## 누구를 위한 서비스인가요?

- AWS를 도입했지만 보안 운영 방법을 모르는 5인 이하 개발팀
- ISMS 인증을 준비 중인데 어디서 시작해야 할지 모르는 팀
- MSSP 비용은 부담스럽고, WAF만으로는 불안한 스타트업

---

## 수익 모델

서버리스 구조(Lambda, API Gateway)를 기반으로 하기 때문에 초기 인프라 비용이 거의 없습니다. 사용자는 트래픽 규모에 따라 쓴 만큼만 지불합니다.

- 월정액 SaaS 구독 (트래픽 규모별 티어)
- ISMS 점검 리포트 자동 생성 서비스
- 전용 AI 모델 파인튜닝 (엔터프라이즈)

---

## 마지막으로

보안은 원래 대기업 것이었습니다.

ShieldOps는 그걸 바꾸려고 합니다. 보안팀이 없어도, 예산이 부족해도, 중소기업이 실시간으로 위협에 대응하고 인증까지 준비할 수 있는 체계를 만드는 것이 목표입니다.
